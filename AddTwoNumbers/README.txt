
# Add Two Numbers #

In this problem, your program should read two whole numbers from the input,
and print out their sum.

// C++
// make sure to first "#include <iostream>"
int a, b;
std::cin >> a >> b;

* Input
The input contains one line, which has two integers
 and, separated by a single space.
 The bounds on these values are 0 a,b 1 000 000

* Output
Output the sum of the two integers
